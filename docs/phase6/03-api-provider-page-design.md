## 3. `/api-connections/new` — 新建 Provider

### 3.1 表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| Provider 类型 | 下拉 (8选项) | ✅ | 选择后自动填充默认 URL |
| 名称 | 文本 | ✅ | 用户自定义标签，最大 50 字 |
| API 地址 | URL | ✅ | 自动填充平台默认值，可修改 |
| API Key | 密码框 | ✅ | 最小 8 字符，AES 加密存储 |
| 模型 ID | 文本 | ✅ | 如 `gpt-4.1`, `claude-sonnet` |
| 设为默认 | 开关 | 否 | |

### 3.2 平台默认值

| Platform | 默认 URL | 默认 Model 提示 |
|----------|---------|----------------|
| OPENAI | `https://api.openai.com` | gpt-4.1 |
| ANTHROPIC | `https://api.anthropic.com` | claude-sonnet-4-20250514 |
| GEMINI | `https://generativelanguage.googleapis.com` | gemini-2.5-pro |
| DEEPSEEK | `https://api.deepseek.com` | deepseek-chat |
| GROK | `https://api.x.ai` | grok-4 |
| CUSTOM_* | （空白） | （空白） |

### 3.3 提交流程

```
填写表单 → 点击 [保存并测试]
  → POST /api/api-configs (创建)
  → 成功后 → POST /api/api-configs/[id]/test (测试连接)
  → 显示结果："连接成功 ✅" 或 "连接失败 ❌ (错误信息)"
  → 返回列表页
```

## 4. `/api-connections/[id]` — 编辑/详情

### 4.1 页面布局

```
┌──────────────────────────────────┐
│  ← 返回      编辑 Provider       │
├──────────────────────────────────┤
│  名称: [My OpenAI___________]    │
│  类型: OpenAI (不可更改)          │
│  地址: [https://api.openai__]    │
│  Key:  [••••••••••] [显示/隐藏]  │
│  模型: [gpt-4.1_____________]    │
│  默认: [ ○ ] 设为默认 Provider    │
│                                  │
│  [ 测试连接 ]  [ 保存 ]          │
│                                  │
│  ───────────────────────────     │
│  [ 删除此 Provider ]  (红色)     │
└──────────────────────────────────┘
```

### 4.2 操作按钮

| 按钮 | 行为 |
|------|------|
| 测试连接 | 调 `POST /api/api-configs/[id]/test`，显示成功/失败 |
| 保存 | 调 `PUT /api/api-configs/[id]` |
| 删除 | 确认弹窗 → `DELETE /api/api-configs/[id]` → 返回列表 |

## 5. 交互约束

| 约束 | 规则 |
|------|------|
| 每用户最多配置数 | FREE: 无硬限制（产品定义允许自备多个 API）；VIP: 无限制 |
| Provider 类型不可更改 | 创建后 platform 不可变（需删除重建） |
| API Key 不可查看 | 编辑时显示占位符，输入新值覆盖旧值 |
| 测试连接超时 | 10 秒 |
| 删除默认 Provider | 允许删除；列表自动无默认状态 |
